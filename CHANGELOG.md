# Change Log

All notable changes to the "markovia" extension will be documented in this file.

## [0.0.1] - 2026-01-23

### Added
- WYSIWYG-style rendering with visible but deemphasized Markdown syntax (inspired by iA Writer)
- Real-time decoration updates as you type
- Heading styling with bold weight and deemphasized # symbols
- Bold/italic/strikethrough formatting with actual text styling and faded markers
- Link rendering with underlined, colored text and faded, italicized URLs
- Code highlighting with background color and faded backticks
- List marker deemphasis for cleaner appearance
- Blockquote styling with italic content and faded > markers
- Code block background highlighting with faded fence markers
- Complete CommonMark formatting support (bold, italic, strikethrough, code)
- Heading commands (H1-H6) with keyboard shortcuts
- List formatting (bullet and numbered lists)
- Blockquote support
- Link and image insertion with URL prompts
- Code block insertion with language specification
- Horizontal rule insertion
- Formatting toolbar at the top of Markdown files (using CodeLens)
- Keyboard shortcuts for all formatting commands (Ctrl/Cmd+Key)
- Configuration options for toolbar and WYSIWYG mode
- Smart toggle behavior (apply/remove formatting with same command)
- Markdown file detection and auto-activation
